# Powerball Predictor

Aplicación web simple que analiza los resultados históricos de Powerball y sugiere números basados en frecuencia.

## ⚠️ Aviso importante
**Este proyecto NO predice números ganadores.** La lotería es un juego de azar. Es solo para fines educativos y de entretenimiento.
